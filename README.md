Структурно схема реалізована на модулі STM32F103 Smart з підключенням до нього TFT LCD 1.3” 240x240 ISP або TFT LCD 1.44 128x128 SPI. 
Для можливості коригування дати та часу до MK STM32F103 лідключено BlueTooth модуль CH-06 з керуванням від Додатку мобільного телефону Android.
Для оброблення параметрів дати і часу використвується Unix - time.
